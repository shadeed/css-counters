CSS Counters Simple Guide

Every time I use CSS Counters, I always read the CSS spec or MDN in order to get a reminder of how they work. In this simple page, you will learn how to memorize them easily.

What are CSS Counters?

They provide us with a way to control number variables in generated content, just like HTML ordered lists.

Steps to add a CSS Counter

  • Step 1: Set the counter

    Pick a name for your counter and add the property counter-reset assigned with the name you picked to the parent element of the list you have.

                  ul { 
                  counter-reset: list; 
              }

  • Step 2: Increment the counter

    Since CSS variables are generated content, we only use them with pseudo elements. Go to the child element you want and then assign counter-increment.

                  li:before { 
                  counter-increment: list; 
              }

  • Step 3: Assign the counter

    Add counter() function to the content property of the pseudo element.

                  li:before { 
                  counter-increment: list; 
                  content: counter(list); 
              }
